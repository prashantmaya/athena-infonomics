### Common Commands Executed

- Backend:
  - `python manage.py migrate`
  - `python manage.py seed_demo_data --file fixtures/seed_demo_data.json`
  - `python manage.py test`
  - `python manage.py runserver`
- Frontend:
  - `npm install`
  - `npm run build`
  - `npm run lint`
  - `npm run dev`
- Docker:
  - `docker compose up --build`
  - `docker compose down`
